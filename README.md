# 📖 About

SoomLog는 개발자를 위한 AI 기반 기술면접 학습 플랫폼입니다.

단순히 면접 질문과 답변을 제공하는 것이 아니라,

- 직접 답변 작성
- AI 피드백
- 모범답안 비교
- 꼬리질문 생성
- 학습 로그 관리

를 통해 실제 기술면접처럼 학습할 수 있도록 설계되었습니다.

> "답을 먼저 보는 것이 아니라,
> **먼저 생각하고 답하는 과정**을 중요하게 생각합니다."

---

# ✨ Key Features

## 🔒 Unlock Learning

답변을 작성하기 전까지

- 모범답안
- AI 피드백
- 심화 설명
- 꼬리 질문

모두 잠겨 있습니다.

답변 제출 후에만 열리도록 설계하여
실제 학습 효과를 높였습니다.

---

## 🤖 AI Feedback

사용자의 답변을 분석하여

- 정확성
- 전달력
- 깊이
- 부족한 개념

을 평가합니다.

예시

```

Score : 88/100

Good
✔ HTTP/2 Multiplexing 설명

Need Improvement
✖ Header Compression 설명 부족

```

---

## 📚 AI Model Answer

난이도별 모범답안을 제공합니다.

- 핵심 답변
- 일반 답변
- 면접 답변
- 심화 답변

---

## 🎯 Tail Questions

답변 이후

AI가 실제 면접처럼

```

HTTP/3는 무엇인가요?

HOL Blocking은 무엇인가요?

HTTP/2는 왜 Binary Protocol을 사용할까요?

```

와 같은 꼬리질문을 생성합니다.
